# じぶんのトリセツ — 実装プラン

## Context

ユーザーが「世の中の些細な困りごとを解決し、隙間時間についつい開いてしまうアプリ」を作りたいという要望から、壁打ちを通じてコンセプトを練り上げた。

**コンセプト**: 武井壮メソッドに着想を得た、パーソナルコンディション記録アプリ。天気・気温・湿度などの外的条件と、気分・エネルギー・パフォーマンスなどの内的状態を記録し、AIが自分だけの「取扱説明書」を生成する。さらに **目標・本番のトラッキング** で日々の練習と結果を紐づけ、**架空のコーチ**（イチロー風、大谷風など）が自分に合ったアドバイスをくれる。

**ターゲット**: アスリート（ゴルフ等）、学習者、営業職、推し活ユーザーなど「毎日の積み重ねで成果を出したい人」。初回セットアップで自分の使い方を選べる。

**ユーザーの選択**:
- アプリ名: **じぶんのトリセツ**
- AI API: **Google Gemini API**
- スコープ: **フルスペック**（記録 + 目標トラッキング + AI分析/コーチング + コミュニティ）
- 技術: 既存プロジェクト（HTML/CSS/JS + Nginx/Docker）の延長
- 初回セットアップで使い方をカスタマイズ可能に
- 架空コーチ設定（「イチローならこう言うかもな？」スタイル）

## 画面構成

### 初回セットアップ（コーチとの対話型オンボーディング）
RPGのキャラメイクのような体験。セットアップ自体が楽しい導入になる。

**Step 1: 使い方の選択**
> 画面: カード形式で4つの選択肢
- 勉強 → サブ選択: 教科、テスト日程入力
- 仕事 → サブ選択: 営業、企画、開発など
- 趣味（スポーツ・ゲーム等） → サブ選択: 種目入力（ゴルフ、ランニング等）
- メンタルヘルス → 気分トラッキング中心の設定

各カテゴリに応じてデフォルトの活動カテゴリやログ項目が自動設定される。

**Step 2: コーチのキャラメイク**
> 画面: RPG風の選択UI
- **性格タイプ**: 情熱的🔥 / 丁寧🌸 / クール❄️ / かわいい🐱
- **関係性**: 年上（先輩/師匠風） / 友達のような（タメ口） / 年下（後輩が応援風）
- **詳細カスタム**（任意）: 一人称、口調などの自由入力欄
- **コーチアバター**: CSSベースのアバター。性格×関係性の組み合わせで色・表情が変わる
  - 情熱的×年上 → 赤系、キリッとした表情
  - かわいい×年下 → ピンク系、にこにこ表情 等

**Step 3: コーチとの対話でセットアップ完了**
> 画面: チャット風UI。コーチがStep2で選んだ口調で話しかけてくる
```
コーチ「はじめまして！あなたのお名前を教えてくれる？」
→ [ユーザー名入力]
コーチ「へー！○○さん！よろしくお願いします！」
コーチ「ところで、私の名前をつけてくれない？」
→ [コーチ名入力]
コーチ「○○かぁ！いい名前だね。○○って呼んでね！」
コーチ「ところで○○さんって、推しとかいる？尊敬する人とか憧れの人！」
→ [推し/尊敬する人 入力]（スキップ可）
コーチ「あー△△か！いいね、彼(彼女)の○○なところいいよね！」（さらっと受け止める程度）
コーチ「じゃあ○○さん、一緒に頑張っていこう！」
```
※ 口調はStep2の選択で変わる（丁寧なら敬語、クールなら短文、かわいいならゆるい語尾）
※ **推しの扱いロジック**:
  - 有名人 → AIがその人のスタイルを踏まえ「△△ならきっとこう言うかもな…」を生成
  - 家族・無名の人 → 格言を捏造せず、一般的な名言を引用（「こういう時に響く言葉があるよ」）
  - スキップ → コーチ自身の言葉のみでアドバイス

**Step 4: 位置情報の許可**
> コーチが「天気も記録すると面白いよ！位置情報を許可してくれる？」と説明

セットアップ完了 → ホーム画面へ。設定は後から変更可能。

### Tab 1: ホーム 🏠
- 今日の天気カード（自動取得）
- 今日のログ状態サマリー
- 「今日のログを記録」CTAボタン
- 連続記録日数（ストリーク）
- 直近のAIインサイト表示
- 次の本番までのカウントダウン（「ラウンドまであと○日」等）
- マイコーチからの一言（キャッシュ済みのアドバイス表示）

### Tab 2: ログ記録 📝
- 日付セレクター（デフォルト: 今日）
- **コンディション記録**:
  - 気分セレクター（5段階: 😫😟😐😊🤩）
  - エネルギーセレクター（5段階バー）
  - 天気カード（Open-Meteo自動取得: 気温・湿度・天候）
- **活動記録セクション**:
  - カテゴリ選択（セットアップで設定したものがデフォルト表示）
  - パフォーマンス評価（★1-5）
  - 数値記録（任意: 練習時間、球数など活動に応じた数値）
  - メモ（テキストエリア）
  - 複数活動追加可能
- **本番記録モード**（トグルで切り替え）:
  - 「これは本番の記録」チェック
  - 結果入力（スコア、点数、成約数など自由記述）
  - 紐づく目標の選択
  - 本番時の特別なコンディションメモ
- 保存ボタン

### Tab 3: 分析 📊
- 期間フィルター（1週間 / 1ヶ月 / 全期間）
- ローカル統計（CSSチャート）:
  - 天気別の平均気分（横棒グラフ）
  - 曜日別パフォーマンス（ヒートマップ）
  - 活動カテゴリ別パフォーマンス
  - **本番結果の推移グラフ**（スコアの変化等）
  - **練習量 vs 本番結果の相関表示**
- 「AIで傾向を分析」ボタン → Gemini API呼び出し
- AI分析結果カード（パターン発見 + トリセツ生成）
  - 「こういう日はパフォーマンスが落ちやすい」
  - 「○○の練習をした週はスコアが良い傾向」
  - 「じぶんのトリセツ」として5つの法則にまとめ
- Gemini APIキー設定（折りたたみセクション）

### Tab 4: コーチ 🎯
- 今日のコンディションサマリー
- **マイコーチカード**（設定したコーチ名 + アイコン表示）
- 「コーチに相談する」ボタン → Gemini API呼び出し
  - 今日の天気・コンディション + 過去の傾向を踏まえたアドバイス
  - **コーチのスタイルで語る**（「イチローならきっとこう言うかもな…」）
  - 具体的な行動提案（「今日みたいな日はアプローチ練習が効果的かも」）
- **目標管理セクション**:
  - 目標登録（「次のラウンドで90切り」「期末テスト数学80点」等）
  - 目標期日設定 → カウントダウン表示
  - 目標達成時の記録と振り返り
- 興味・推し設定（タグ形式）
  - 落ち込んだ時の推しコンテンツおすすめ生成
- イベント登録（新曲リリース、大会日程など）

### Tab 5: コミュニティ 👥
- Firebase設定ウィザード（未設定時はガイド表示）
- 匿名認証 / Google認証
- 地域の気分マップ（今日のエリア平均）
- 共有設定トグル
- 似た傾向のユーザー表示

**重要**: Firebase未設定でも他の4タブは完全動作する。コミュニティはオプション機能。

## API統合

### Open-Meteo（天気）
- 既存 index.html のパターンを再利用（`/home/user/test/index.html:116-134` のWMOコード含む）
- `navigator.geolocation` → フォールバック: 廿日市市(34.35, 132.33)
- 湿度取得のため `hourly=relativehumidity_2m` パラメータ追加
- 30分キャッシュ

### Gemini API（AI分析・コーチング）
- エンドポイント: `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent`
- ユーザーが自分のAPIキーを入力、LocalStorageに保存
- **分析プロンプト**: ログデータをJSON化し、以下を依頼:
  - 天気×気分の相関、曜日傾向
  - 練習内容 × 本番結果のパターン発見
  - 「じぶんのトリセツ」として5つの法則にまとめ
- **コーチングプロンプト**: マイコーチのペルソナを含めて生成:
  - 「あなたは{コーチ名}のような{スタイル}なコーチです」
  - 今日の天気 + 過去の傾向 + 次の本番までの日数を踏まえたアドバイス
  - 「{コーチ名}ならきっとこう言うかもな…」スタイルで出力
  - 推し/興味情報を元にメンタルケア提案も
- レート制限: ボタン押下後10秒間無効化

### Firebase（コミュニティ）
- CDN経由でcompat SDKをロード（ビルドツール不要）
- ユーザーが自分のFirebase設定を入力（セットアップウィザード）
- 匿名認証をデフォルト、Google認証はオプション
