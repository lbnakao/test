# reference/

過去投稿のキャプション本文をここに保存します。Claude Code が文体の見本として読み込みます。

## 必要なファイル

以下2ファイルを用意してください（PROMPT.md / CLAUDE.md から参照されています）：

- `post_DXg4mJ8gX5G.txt`
- `post_DVedtvKgT8R.txt`

ファイル名末尾の `DXg4mJ8gX5G` などは Instagram の投稿コードです（`https://www.instagram.com/p/DXg4mJ8gX5G/` の `p/` の後ろの部分）。文体の参考にしたい投稿のキャプションをコピーして UTF-8 で保存してください。

参考にしたい投稿を増やしたい場合は、`post_<コード>.txt` というファイル名で追加し、`PROMPT.md` / `CLAUDE.md` の参照リストも更新してください。
